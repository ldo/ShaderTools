 AIDE A L'UTILISATION DE L'ADD-ON

 Ouvrir... - Comment ca marche ?

Ce module vous permet d'acceder a votre base de materiaux. Une fois avoir cliquer sur le module vous devrez cliquer sur le Bookmarks nouvellement cree "Base Materiaux". Vous pouvez activer le mode icones pour voir le contenu sous forme d'icones. Le bouton "RECHERCHER" vous permet d'effectuer une recherche dans la base. Pour afficher tout les materiaux apres une recherche saisissez une valeur vide dans la module de recherche et valider. Le module historique vous permettra d'acceder aux 20 derniers materiaux utilises. Une fois votre choix effectue double cliquez sur le materiau pour vous permettre de l'importer sur votre object actif.

 

 Sauvegarder - C'est quoi ?

Ce module vous permet de sauvegarder le materiau actif directement dans la base. Il vous suffit pour l'utiliser de completer les champs (en partie ou totallement) et de valider par le bouton "OK". Vous pourrez reutiliser ce materiau en passant par le module "Ouvrir…".

 

 Exporter - Pour quelle raison ?

Ce module vous permet d'exporter un materiau sous forme de fichier executable par n'importe quel autre Blender. Pour Exporter il vous suffit de cliquer sur le module puis de selectionner le repertoire, le nom du fichier ainsi que le nom du createur (c'est a dire VOUS), puis de valider. Pour executer ce fichier il vous suffira d'ouvrir le "Texteditor", puis d'aller sur le menu "Text" puis "Open text block" et de charger le fichier exporter. Ne vous restera plus qu'a cliquer sur le bouton "Run Script" du "Text editor".

 

 Importer - Euh! mais encore?

Ce module vous permet d'importer un materiau sous forme de fichier executable dont le format reconnut est les format "blex". Apres avoir exporter votre materiau, cliquez sur "Importer" puis selectionnez lefichier blex puis validez votre choix pour importer votre materiau.

 

 Nouveau - A quoi ça sert ?

Ce module va vous permettre de charger un modele de materiau. Vous pourrez jongler entre les quatres premiers calques suivant que vous ayez besoin d'une base ressemblant a ne Sphere/Cube/Plan/Suzanne. Ce modele est celui utiliser pour afficher les previews de la base de materiaux. Il est pratique pour la creation d'un nouveau materiau.

 

 Configuration - Je dois y toucher ?

Ce module permet de parametrer l'endroit où se situe la base de donnees de materiau. Pratique pour les personnes desireuses de modifier l'emplacement de la base.
