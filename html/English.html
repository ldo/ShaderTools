 HOW TO USE THIS ADD-ON

 Open... - How does it work ?

This module allows you to obtain your files of materials. Once to have to click the module you will have to click Bookmarks, recently create " Material Database ". You can activate the "mode icons" to see the contents of the icons form. The button "to LOOK FOR" allows you to make a search(research) in the data/file. To show/see everything ... after searched materials, take an empty value in the module of search and validate. The historic module will allow you to reach the last 20 materials use. Once your choice makes double click the material to allow you of the importer on your active object.

 

 Save - What is it?

This module allows you to protect the active material directly in the base. It is enough to you to use him(it) to complete fields (partially or totally) and to validate by the button "OK". You can re-use this material by way of the module "Open".

 

 Export - Why ?

This module allows you to export a material in the form of feasible file by any other Blender. For Exporter you just have to click the module then of the directory, the name of the file as well as the name of the creator (it is has to say YOU), then to validate. To execute this file you will just have to open "Texteditor", puis d'aller sur le menu "Text" puis "Open text block" et de charger le fichier exporter. Ne vous restera plus qu'a cliquer sur le bouton "Run Script" du "Text editor".Then to go on the menu " Text " then " Open text block " and to ask the file to export. Will not stay in you more than has to click the button " Run Script " of " Text editor ".

 

 Import - Woops! but still ?

This module enables you to import a material in the form of executable file whose format recognized is the format "blex". After having to export your material, click on "Importing" then select blex filethen validateyour choice.

 

 New - To what that is of use ?

This module is going to allow you to load(charge) a model of material. You can juggle between the first four copies as you need a looking like base has Sphere Sphere/Cube/Plan/Suzanne Sphere/Cube/Plan/Suzanne Sphere/Cube/Plan/Suzanne Sph. This modele is the one to use to post(show) the previews of the base of materials. He(it) is practical for the creation of a material new.

 

 Configuration - I have to get(touch) it ?

This module allows to parameter the place where is situated the database of materials. Practice for the desirous persons to modify the place of the base.
