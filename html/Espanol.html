 AYUDA A LA UTILIZACION DEL MODULO

 Abrir ... - ¿Cómo funciona?

Este módulo le permite acceder a sus materiales de base. Una vez que haga clic en el módulo, debe hacer clic en el recién creado marcadores "Materiales de Base." Puede activar los iconos de los contenidos en forma de iconos. El botón "Buscar" le permite buscar en la base de datos. Para ver todos los materiales entran en una búsqueda de un valor vacío en el módulo de búsqueda y confirmar. El módulo de la historia le dará acceso a los últimos 20 materiales utilizados. Una vez que su elección realiza doble clic en el material para que puedas importar el objeto activo.

 

 Salvar - ¿Qué es?

Este módulo le permite guardar el material activo directamente en la base de datos. Todo lo que necesitas para usar, rellene los campos (en parte o totalmente) y confirme con "OK". Puede volver a utilizar este material a través del módulo "Abrir ...".

 

 Exportación - ¿Por qué?

Este módulo le permite exportar un material como un archivo ejecutable por cualquier otro Blender. Exportar a un simple clic en el módulo y seleccione el directorio, nombre de archivo y el nombre del creador (es decir), y valide. Para ejecutar este archivo, basta con abrir la "TextEditor", y luego ir al menú "Texto" y después "bloque de texto abierto" y descargar el archivo de exportación. ¿Tienes que hacer clic en "Ejecutar secuencia de comandos" en el "Editor de texto".

 

 Importer - Euh! mais encore?

Este módulo le permite importar una forma material de archivo ejecutable, cuyo formato es el formato reconocido "Blex". Después de exportar el material, haga clic en "Importar" y seleccionar somefile Blex y confirmar su elección para importar su material.

 

 Nuevo - ¿Cuál es el punto?

Este módulo le permitirá cargar un modelo de material. Puede alternar entre los primeros cuatro capas, dependiendo de si usted necesita una base como un no Esfera / Cubo / Plan / Suzanne. Este modelo se utiliza para mostrar vistas previas de los materiales base. Es conveniente para la creación de un nuevo material.

 

 Configuración - lo toco?

Este módulo le permite establecer dónde está la base de datos de material. Conveniente para la gente que quiere cambiar la ubicación de la base.
